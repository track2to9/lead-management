# Prospect 상세 페이지 — 6탭 리치 결과물 리디자인

> 고객이 "직접 찾는 것보다 낫다"고 느낄 정도의 정보 밀도. 결과물 풍성함이 매출 포인트.

## 현재 상태

현재 prospect 상세 페이지는 단일 페이지에 모든 정보를 나열하는 평면적 구조. 원문 증거(스크린샷, 외국어 텍스트)가 부족하고, 정보가 빈약하게 느껴짐.

## 변경 사항

단일 페이지 → **6탭 구조**로 전환. 각 탭이 독립적인 가치를 제공하고, 원본 증거 + 한국어 해석이 항상 함께 표시.

## 6탭 구조

### 탭 1: 개요
- 회사명 + 종합 점수 뱃지 + 등급(HIGH/MEDIUM/LOW) + 분류(구매자/경쟁사)
- 회사 요약 (한국어, 2-3문장)
- 접근 전략 카드 (강조 배경)
- 5차원 점수 바 차트 (제품적합도/구매시그널/기업역량/접근성/전략가치) — 우측 패널
- 승인/제외/추가분석 액션 버튼 + 피드백 입력

### 탭 2: 홈페이지 분석
- **상단**: 홈페이지 전체 스크린샷 (클릭 시 모달 확대) + 사이트 기본 정보 (URL, 언어, 이메일, 전화, 위치, SNS 링크)
- **중간**: 핵심 발견 카드 N건
  - 각 카드: 왼쪽 원문(폴란드어 등, 이탤릭) + 오른쪽 한국어 번역(굵게)
  - 하단: 매칭 관련성 분석 + 관련 점수 차원 태그 (제품적합도↑ 등)
- **하단**: AI 종합 분석 + 발견된 제품/서비스 태그 (매칭 제품 강조 표시)

### 탭 3: SNS 활동
- **채널 요약 바**: 발견된 SNS 채널별 뱃지 (LinkedIn/Facebook/Instagram/X) + 팔로워 수
- **게시글 카드**: 최근 3-6개월 게시글
  - 스크린샷 썸네일 + 채널 태그 + 날짜
  - 원문 (이탤릭, 외국어) + 한국어 번역 (강조)
  - 구매 시그널 하이라이트 (관련 제품 언급 시 특별 강조)
- **하단**: AI SNS 활동 분석 요약 (포스팅 빈도, 주요 관심사, 구매 시그널 평가)

### 탭 4: 뉴스 & 전시회
- **2컬럼 레이아웃**
- **왼쪽: 관련 뉴스**
  - 뉴스 카드: 원문 제목 + 한국어 번역 + 출처 + 날짜 + 원본 링크
- **오른쪽: 관련 전시회**
  - 전시회 카드: 이름, 장소, 시기, 관련성 설명, 추천 액션

### 탭 5: 연락 전략
- **이메일 초안**: 제목 + 본문 (복사 가능한 형태)
- **팔로업 시퀀스**: Day 3 / Day 7 / Day 14 카드 (각각 다른 색상)
  - 각 카드: 단계명 + 전략 + 제목 + 본문 미리보기

### 탭 6: 바이어 인텔리전스
- **4칸 그리드**: 현재 공급업체 (태그), 회사 규모, 의사결정권자, 최적 접근 타이밍
- **경쟁 환경 분석**: 전체 너비 카드, 진입 가능성과 전략 포함

## 데이터 흐름

- 탭 1 (개요): `prospects` 테이블의 기존 필드 + `score_breakdown`
- 탭 2 (홈페이지): `evidence` 테이블 (source_type = 'website') + prospects의 evidence_quotes
- 탭 3 (SNS): `evidence` 테이블 (source_type = 'linkedin'/'facebook'/'instagram'/'x')
- 탭 4 (뉴스&전시회): `evidence` 테이블 (source_type = 'news') + `exhibitions` 테이블
- 탭 5 (연락전략): prospects의 email_subject/email_body/followup_sequence
- 탭 6 (인텔리전스): prospects의 current_suppliers/company_size/decision_maker/best_timing/competitive_landscape

## 포털 변경 범위

- `portal/src/app/dashboard/project/[id]/prospect/[pid]/page.tsx` — 전면 재작성 (6탭 구조)
- 필요 시 탭별 서브 컴포넌트 분리

## 포털 외 변경 없음

파이프라인, DB 스키마는 변경하지 않음. 기존에 수집된 데이터를 탭별로 재배치하여 표시하는 것이 이번 범위.
